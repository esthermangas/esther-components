## Width and Height

You can change width and height.

## Square

This prop changes the border type of the Surface. By default it is round.

<Preview>
    <Surface height="150px" width="200px" >
        When the prop square is false, the border is round.
    </Surface>
    <Surface height="150px" width="200px" square>
        When the prop square is true, the border is square.
    </Surface>
</Preview>

## Variant

You can change the style of the component with this prop. There are three types, plain, outlined and elevation.

<Preview>
    <Surface height="150px" width="200px">
        Plain variant.
    </Surface>
    <Surface height="150px" width="200px" variant="outlined">
        Outlined variant.
    </Surface>
    <Surface height="150px" width="200px" variant="elevation">
        Elevation variant.
    </Surface>
</Preview>
