# TextField

<Props of={TextField} />

## FullWidth

This property changes the width of the TextField allowing it to fill up to its parent width.

<Preview>
    <TextField label="This is a Full TextField" fullWidth />
</Preview>

## Color

You can change the color of the TextField with this property, but it only accepts one of our pre-defined colors that you can find [here](/?path=/docs/colors--page).

<Preview>
    <TextField label="This is a estherGreen TextField" value="greenEsther" />
    <TextField label="This is a cadetBlue TextField" color="cadetBlue"  value="cadetBlue" />
</Preview>

## OnChange

This is a function in charge to pick up the information saved in the inputs.

## Error

This property is used to show that there is an error.

<Preview>
    <TextField label="This is an error example" error />
</Preview>

## Password

This property is used to hidden passwords.

<Preview>
    <TextField label="Password"  password />
</Preview>

## InfoMessage

This property is used to add information messages.

<Preview>
    <TextField label="Example infoMessage"  infoMessage="This is the message" />
</Preview>
