# Text

<Props of={Text} />

## tagHtml

This property changes the final rendered HTML tag.

<Preview>
          <div>
              <Text tagHtml='h1' >
                  This is a h1 title
              </Text>
              <div>
                  <Text display='inline-block'>
                        This is a paragraph.
                  </Text>
                  <Text display='inline-block' tagHtml='span'>
                        This is a span.
                  </Text>
              </div>
          </div>
</Preview>

## Color

You can change the color of the TextField with this property, but it only accepts one of our pre-defined colors that you can find [here](/?path=/docs/colors--page).
The component TextField only applies the color property when the tagHtml is set to a "h" variant. Eg: h1 or h2.

<Preview>
          <div>
              <Text tagHtml="h1" color='estherGreen' >
                  This is a h1 title
              </Text>
              <Text tagHtml="h3" color="cadetBlue">
                    This is a h3 title.
              </Text>
          </div>
</Preview>
