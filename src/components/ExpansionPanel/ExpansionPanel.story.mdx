## Color

You can change the color of the ExpansionPanel with this property, but it only accepts one of our pre-defined colors that you can find [here](/?path=/docs/colors--page).

## Open

This property controls the state of the component opening or closing it.

## OnChange

You can pass a callback function to this prop and it will be called every time the panel is opened or closed and it will receive the state as the first parameter.
