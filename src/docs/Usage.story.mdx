# Usage

This is Esther Manga's custom react components library

## Install

Install it using your favorite bundler.


bash
npm install --save bloobirds-platform-component-library

bash
yarn add bloobirds-platform-component-library

## Import

To use it, simply import the components. All of them are accesible from the root, so you don't need to navigate towards them.
